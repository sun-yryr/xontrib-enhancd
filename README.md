xontrib-enhancd
===============================

[b4b4r07/enhancd](https://github.com/b4b4r07/enhancd) for xonsh

Installation
------------

To install use pip:

    $ xpip install xontrib-enhancd


Or clone the repo:

    $ pip install git+https://github.com/sun-yryr/enhancd

Usage
-----

in xonsh

    $ xontrib load enhancd
